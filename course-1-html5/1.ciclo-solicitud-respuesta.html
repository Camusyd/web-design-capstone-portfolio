El Misterio Resuelto: Ciclo de la Comunicación Web

¿Qué ocurre cuando escribimos una URL?

Fase 1: La Solicitud (Request)

Cuando un usuario escribe una URL y presiona Enter, el navegador (el **cliente**) envía una **solicitud HTTP** al servidor. Esta solicitud pregunta al servidor: "Por favor, dame el archivo que tienes en esta dirección."

Fase 2: La Respuesta (Response)

El **servidor** encuentra el archivo (en este caso, un archivo **.html**) y lo envía de vuelta al navegador junto con una **respuesta HTTP** (como el código 200 OK).

Fase 3: El Renderizado

El navegador recibe el código HTML, lo lee línea por línea, construye el **Modelo de Objetos del Documento (DOM)** y, finalmente, **renderiza** la página web visible.

Archivos que Recibe el Navegador:

  • **HTML:** La estructura de la página.
  • **CSS:** Los estilos (cómo se ve).
  • **JavaScript:** La interactividad (lo que hace).
