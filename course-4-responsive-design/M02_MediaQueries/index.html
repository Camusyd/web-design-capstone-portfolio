Módulo 2: Consultas de Medios y Breakpoints

Diseño Adaptativo con @media

El diseño base se ve en móvil. Al redimensionar la ventana (o ver en una pantalla más grande), el CSS utiliza Media Queries con **min-width** para aplicar nuevos estilos.

Tarjeta 1: Mobile-First

En móvil, estas tarjetas se apilan (estilo base). Al llegar al breakpoint de Tablet, se pondrán una al lado de la otra.

Tarjeta 2: Breakpoint Tablet

Al crecer la pantalla, las Media Queries se activan para crear un layout de dos columnas, manteniendo la consistencia visual.

Tarjeta 3: Breakpoint Desktop

Al llegar al breakpoint de escritorio, el layout cambiará a tres columnas para aprovechar mejor el espacio horizontal.

Abre el inspector (F12) y arrastra el borde para ver los cambios. (cambia el ancho de la pagina)
