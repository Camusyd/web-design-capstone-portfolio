Responsive Showcase: La Web Moderna

Integrando **Mobile-First**, **A11y**, **Grid** y **Bootstrap** en un diseño unificado.

DEMO

Principios Fundamentales

Mobile-First

La base del diseño responsivo moderno. Comenzamos con estilos para el móvil (clase `col-12`) y luego **añadimos** complejidad con `min-width` para pantallas grandes. **Mejora el rendimiento.**

Media Queries

Permiten cambiar el layout y los estilos en puntos clave (**breakpoints**). Bootstrap usa `min-width` (ej., `col-md-6`) para activar estos estilos en resoluciones mayores a 768px.

Unidades Fluidas

Uso de **`rem`** (tipografía) y **`%`** (ancho de contenedores) en lugar de píxeles fijos. Garantiza que el contenido y el espaciado escalen proporcionalmente al tamaño del *viewport*.

Maquetación Avanzada: Grid & A11y

CSS Grid y Flexbox

El sistema de **Grid de Bootstrap** es una implementación práctica de **CSS Grid**. Aquí, el layout utiliza `row` y `col-md-*` para dividir el espacio. Dentro, usamos Flexbox (`d-flex`) para la alineación perfecta de la imagen y el texto, demostrando una **maquetación mixta** profesional.

Accesibilidad Dinámica

El **Toggle de Modo Oscuro** en la Navbar es un ejemplo de las Media Queries de accesibilidad (`prefers-color-scheme`) controladas manualmente con JavaScript para ofrecer una opción de alto contraste o visualización nocturna al usuario.

Ejemplos Interactivos (Bootstrap JS)

Componentes con Funcionalidad

El carrusel es 100% responsivo y viene con ARIA/accesibilidad integrada.

Navbar Plegable

Usa **JavaScript de Bootstrap** para plegarse en móvil, sin escribir una línea de JS propio.

Scroll Spy

Resalta automáticamente el enlace activo en la navegación al desplazarte (Atributo `data-bs-spy`).

Prueba de Usabilidad JS y Contacto

El contenido largo a continuación es para probar dos funcionalidades clave de usabilidad:

  1. El **Scroll Spy** de Bootstrap: Mira cómo los enlaces en la barra de navegación se activan a medida que esta sección (`#contacto`) entra en el viewport.
  2. El botón **Volver Arriba** (nuestro JS personalizado): Aparecerá abajo a la derecha una vez que hayas bajado lo suficiente, con un desplazamiento suave (`behavior: 'smooth'`).
